dotvim
======

my .vim &amp; .vimrc

install
======


* ```git clone https://github.com/atttx123/dotvim.git ~/.vim```
* ```ln -s ~/.vim/vimrc ~/.vimrc```
* ```ln -s ~/.vim/ideavimrc ~/.ideavimrc```

dependency
======

* `pip install jedi flake8 autopep8`

* (*highly recommended*) install: https://github.com/universal-ctags/ctags
