dotvim
======

my .vim &amp; .vimrc

install
======

* ```git clone https://github.com/atttx123/dotvim.git ~/.vim```

* ```git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/vundle```

* Launch vim and run ```:PluginInstall```

  > To install from command line: ```vim +PluginInstall +qall```
